# NEXUS GLASS WINDOWS

## Purpose

Every major Nexus object should open into a glass workspace.

A glass window is not a data-entry form.
A glass window is not a traditional CRM screen.
A glass window is not a report.

A glass window answers:

- What is this?
- Who is involved?
- What is happening?
- What is next?
- What is blocked?

The user should always be able to understand the situation and take action without navigating through multiple modules.

## Universal Layout

All glass windows should follow the same structure.

### Header

Identity of the object.

Examples:

- Lead name
- Opportunity name
- Site name
- Project name
- Work order name

### Overview

High-level summary.

Examples:

- Stage
- Status
- Owner
- Priority
- Created date
- Last activity

### Related Information

Connected people, sites, files, tasks, activities, surveys, and documents.

### Next Best Actions

The most important section.

Examples:

- Call
- Email
- Schedule follow-up
- Create opportunity
- Create project
- Create work order
- Generate quote
- Run ARIA

The user should never wonder what to do next.

## Person Window

Purpose:

Understand everything related to a person.

Sections:

- Overview
- Organizations
- Sites
- Opportunities
- Projects
- Activities
- Files

Questions answered:

- Who is this?
- What are they responsible for?
- What are we working on together?

Sections:

- Projects
- Contracts
- Work Orders
- Assets
- Surveys
- Invoices
- Files

Questions answered:

- What is happening at this site?

- What revenue is attached?

## Knowledge Surface

Every glass window should automatically surface relevant knowledge.

Examples:

- Survey Images
- Attachments
- Document Templates
- Product Manuals
- Manual Chunks
- KB Articles

Knowledge should appear inside the workflow.

Users should not need to leave the glass window to find information.

## Design Rule

Users should never think:

- Which module do I open?
- Which screen do I use?
- Where is that file?

Users should think:

- Open the object.
- Understand the situation.
- Take the next action.
